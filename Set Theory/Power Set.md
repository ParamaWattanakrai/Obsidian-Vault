Is the set containing all subsets of a given set.
Let $S$ be a set.
$$
\begin{flalign}
P(S)=\{x|x\subseteq S\}&&
\end{flalign}
$$
"The power set of $S$ is $x$ such that $x$ is a subset of $S$"
If $|S|$ is finite then:
$|P(S)|=2^{|S|}$

Example:
Let $A=\{1,2,3\}$
$P(A)=\{\emptyset,\{1\},\{2\},\{3\},\{1,2\},\{1,3\},\{2,3\},\{1,2,3\}\}$
$|P(A)|=2^{|A|}=2^3=8$
