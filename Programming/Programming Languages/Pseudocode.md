# Conditional Statements
a.
```pseudocode
if (condition)
	then statement_1
	else statement_2
```
b.


# Iterative Statements

## While Statement
```pseudocode
while (condition)
	body_statements
end while
```

## For Statement
```pseudocode
for variable := initial_expression to final_expression
	body_statements
next variable
```
